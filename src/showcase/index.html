stock-elements

Catalog

Framework-agnostic web components for the Stockroom family. Each element ships with a React adapter and a documented CSS custom property theming surface.

Loading Bar Loading Grid Select

Loading Bar

<sce-loading-bar>

Terminal-emulator-style pixelated progress indicator with three pre-baked themes. Accepts real progress polling or indeterminate "fake" mode for open-ended work.

Loading Grid

<sce-loading-grid>

The bar's 2D sibling. Same block cell, same three themes, same randomness / speed / fakeMode / progress rules — laid out as a grid that auto-fits its container on both axes. Use it as a spinner replacement inside cards, tiles, or any square / tall slot where a long horizontal bar wouldn't fit.

Select

<sce-select>

Searchable single-select combobox that always clamps its panel to the visible viewport and scrolls internally — a drop-in replacement for native <select> or Radix Select when the option list may exceed screen height. Keyboard-first: ↑ ↓ navigate, ↵ select, esc close, type anywhere to filter.
